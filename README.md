# 🔢 Cálculo de Soma e Média com Vetores em Java

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que permite ao usuário inserir uma quantidade de números, armazená-los em um vetor e, ao final, calcular a soma e a média desses valores.

O programa também exibe todos os números digitados, proporcionando uma visualização completa dos dados inseridos.

---

## 🚀 Funcionalidades

* Leitura de múltiplos números informados pelo usuário
* Armazenamento dos valores em um vetor
* Cálculo da soma total dos números
* Cálculo da média dos valores
* Exibição dos números digitados

---

## 🧱 Estrutura do Código

O programa utiliza:

* Vetor (`array`) para armazenar os números digitados
* Estruturas de repetição (`for`) para leitura e processamento
* Variáveis auxiliares para cálculo da soma e média

---

## 🖥️ Tecnologias Utilizadas

* Java
* Programação estruturada
* Entrada de dados com `Scanner`

---

## 📚 Conceitos Aplicados

* Vetores (arrays)
* Estruturas de repetição (`for`)
* Operações matemáticas (soma e média)
* Entrada e saída de dados
